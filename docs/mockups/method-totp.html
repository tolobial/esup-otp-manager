Instructions

  1. 1

    Téléchargez une application comme Google Authenticator, FreeOTP ou Authy.

  2. 2

    Scannez le code QR ou saisissez la clé manuellement.

  3. 3

    Entrez le code à 6 chiffres affiché dans l'application pour valider.
